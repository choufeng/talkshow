功能规格

左侧菜单栏

  • 宽度：160px
  • 包含：首页、设置
  • 当前选中项高亮显示
  • 支持后续扩展更多菜单项

快捷键设置

  • 点击"修改"进入录制状态
  • 按下组合键完成设置
  • 按 Esc 取消录制
  • 自动保存到 config.json

技术实现

前端 (SvelteKit)

  • 使用 Svelte 5 的 runes 语法 ($state, $effect)
  • 创建 Settings 页面组件
  • 实现快捷键录制组件 (ShortcutRecorder)
  • 通过 Tauri IPC 调用后端 API

后端 (Tauri/Rust)

  • 添加 Tauri 命令：get_config, save_config
  • 添加 Tauri 命令：register_shortcut, unregister_shortcut
  • 修改 config 模块支持动态快捷键更新
