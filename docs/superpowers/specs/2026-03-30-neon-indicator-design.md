# 录音浮动窗霓虹科幻风改造设计

## 背景

当前录音浮动窗 (`/recording`) 使用简单的红色脉冲点和 CSS 旋转圆环指示录音/处理状态。用户希望提升视觉品质，改为 Cyberpunk 霓虹风格。

## 涉及文件

- `src/routes/recording/+page.svelte` — 浮动窗 UI 组件（模板 + 逻辑 + 样式，213 行）
- `src-tauri/src/lib.rs` — 窗口创建参数（可能需调整窗口尺寸）

## 设计方案：霓虹脉冲 + 发光水母

### 整体风格

- **视觉语言：** Cyberpunk 霓虹，深色背景 + 高对比发光元素
- **实现方式：** 纯 SVG 动画（无外部动画库依赖），CSS `filter: drop-shadow` 实现辉光
- **容器保持不变：** 药丸形、毛玻璃背景 (`rgba(30,30,30,0.92)` + `backdrop-filter: blur(20px)`)、取消按钮

### Phase 1：录音中 (Recording)

用 SVG 替换当前 CSS 脉冲点：

- **脉冲环：** 品红 (`#ff0055`) 圆环，半径 7→9→7 呼吸动画，同步透明度 1→0.5→1，周期 1.2s
- **核心点：** 品红实心小圆，固定不动，与环叠加
- **辉光：** `filter: drop-shadow` 实现霓虹辉光效果
- **计时器：** 等宽字体，`#ff0055` 颜色，`text-shadow: 0 0 6px rgba(255,0,85,0.4)` 文字辉光
- **REC 标签：** 小号等宽字体，低透明度品红色

布局：`[脉冲环SVG] 00:12  REC`

### Phase 2：处理中 (Processing)

用发光水母 SVG 动画替换当前 CSS spinner：

**水母结构：**
- **钟体 (Bell)：** 半透明渐变填充 (`radialGradient`，中心 `#00c8ff` opacity 0.3 → 边缘 0.05) + `#00c8ff` 描边 1.2px
- **钟体动画：** 路径变形实现收缩/舒张，周期 2s
- **内部器官：** 一个半透明椭圆，随钟体收缩联动
- **触手：** 5 根，长度从外到内递增
  - 外侧两根：较短，opacity 0.5，周期 2.5s
  - 中间两根：中等，opacity 0.7，周期 2.2s
  - 中央一根：最长最亮，opacity 0.8，周期 2s
- **触手动画：** 每根独立的贝塞尔曲线路径变形，模拟飘动
- **整体浮动：** `translate Y: 0 → -2px → 0`，周期 2s
- **辉光：** SVG `filter` with `feGaussianBlur stdDeviation=2.5` (钟体) / `1.5` (触手)

布局：`[水母SVG] 处理中`

**颜色方案：**
- 水母主色：`#00c8ff` (青色)
- 容器背景微调：`rgba(0,200,255,0.08)` + `border: 1px solid rgba(0,200,255,0.2)`
- 文字：`#00c8ff`，带 `text-shadow: 0 0 6px rgba(0,200,255,0.4)` 辉光

### 取消按钮

保持现有样式不变（圆形按钮，hover 变红）。

### 窗口参数

可能需要从 160x44px 调整到 180x48px，给水母 SVG 更多空间。具体在实现时根据视觉效果微调。

## 不做的事

- 不引入 framer-motion 或其他动画库
- 不改变 Tauri 事件通信流程
- 不改变窗口位置、透明度等基础属性
- 不修改 Rust 后端代码（除非需要调窗口尺寸）
