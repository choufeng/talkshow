# TalkShow 系统图标设计

日期: 2026-03-30

## 概述

为 TalkShow 语音输入工具设计全新的系统图标体系，替换当前 Tauri 默认图标。设计覆盖 macOS 应用图标、系统托盘图标、Windows 图标等全平台场景。

## 设计决策

- **核心元素**: 麦克风（直观表达语音输入）
- **风格**: 几何图形风（用几何形状拼出轮廓，现代感）
- **主色调**: 绿色系（`#30D158 → #28B44D` 渐变）
- **形态**: 全填充式圆角矩形（macOS 标准 squircle）
- **辅助元素**: 圆点粒子声波（6 个白色圆点，对称分布）

## 主图标（App Icon）

### 底板

- 形状: 圆角矩形，全填充式
- 圆角: macOS 标准 squircle（约 22% 圆角率）
- 填充: 线性渐变 `#30D158 → #28B44D`，方向 160°（左上到右下）

### 图形元素（基于 160px 画布）

**麦克风头部**

- 形状: 白色圆角矩形，56×70px，居中偏上
- 位置: x=52, y=18
- 顶部全圆角（rx=28）
- 不透明度: 95%

**麦克风支架**

- 白色弧线，7px 粗细，圆角端点（stroke-linecap: round）
- 路径: 从左侧 (42, 82) 经过底部弧线到右侧 (118, 82)，弧线最高点约 y=115
- 不透明度: 95%

**底部结构**

- 竖线: (80, 108) → (80, 135)，白色，7px，圆角端点，95% 不透明度
- 横线: (62, 135) → (98, 135)，白色，7px，圆角端点，95% 不透明度

**粒子声波（圆点）**

左侧（由近到远）:

| 位置 (cx, cy) | 半径 | 不透明度 |
|---------------|------|---------|
| (24, 68)      | 7    | 55%     |
| (12, 82)      | 5    | 35%     |
| (8, 60)       | 4    | 22%     |

右侧: 与左侧水平镜像对称

- 颜色: 白色
- 效果: 由近到远，半径和不透明度递减，营造声音扩散感

## 系统托盘图标（macOS Tray）

### 空闲状态

- 纯黑色线条剪影（macOS 自动处理为匹配系统主题）
- 仅保留麦克风核心形状（头部+支架+底部），去除粒子
- 尺寸: 22×22px (1x) / 44×44px (2x Retina)

### 录音状态

- 同空闲状态剪影
- 右上角添加红色实心小圆点指示器
- 圆点颜色: `#FF3B30`（macOS 系统红）
- 圆点大小: 约 6px (1x) / 12px (2x)

## 尺寸简化规则

| 尺寸范围       | 策略                                    |
|--------------|-----------------------------------------|
| ≥128px       | 完整设计（麦克风 + 粒子 + 支架 + 底部）    |
| 64-127px     | 保留麦克风 + 粒子，线条适当加粗           |
| 32-63px      | 仅保留麦克风头部 + 支架，去除粒子          |
| ≤31px        | 仅麦克风头部剪影                          |

## 设计原则

1. **纯扁平化** — 无阴影、无内发光、无 3D 效果
2. **几何简洁** — 所有元素由基本几何形状构成
3. **可缩放** — 从 22px 托盘到 1024px 应用图标均保持可辨识
4. **平台适配** — 遵循 macOS squircle 规范，Windows 磁贴规范
5. **状态区分** — 空闲和录音状态有明确视觉区分
